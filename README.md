# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/16RJ-WLwLEXMomyg40WKUjXSc1SAsfs_-

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (Optional) Set the `GEMINI_API_KEY` in `.env.local`, or enter your key directly in the app via "Set API Key"
3. Run the app:
   `npm run dev`
